# Checkpoint-4-Group-17
This is a Node.JS front-end.

## Installation
Install the following softwares/frameworks.
- MariaDB or MySQL
- Node.JS

Run `npm install` in the root directory.
Then run `npm start`.

All code is hand-written; no autogenerated or automated processes.
